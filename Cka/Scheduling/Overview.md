# Kubernetes Scheduling Overview

### Manual Scheduling
- Schedule Pods manually without the default scheduler

### DaemonSets
- Run Pods automatically on all or selected nodes

### Labels and Selectors
- Organize and identify Kubernetes objects
- Used for grouping and scheduling decisions

### Resource Requests and Limits
- Define CPU and memory requirements
- Influence Pod scheduling behavior

### Multiple Schedulers
- Configure and use custom schedulers
- Run more than one scheduler in a cluster

### Scheduler Events
- View scheduling-related events
- Troubleshoot scheduling issues
